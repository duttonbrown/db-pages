What this is

This is a plain index.html file living at marketing/hb/example-page/index.html. When Thomas pushes the repo, it goes live at the URL in the footer below.

How to make your own

  1. Make a new folder inside marketing/hb/ — e.g. q2-report
  2. Inside it, create a file called index.html
  3. Copy this file as a starting point, or write your own from scratch
  4. Save the file and open it in your browser (double-click) to preview
  5. Tell Thomas to push when you want it live

Tip

The filename matters. Always name the main file index.html — that's the file the browser loads when someone visits the folder URL. If you call it anything else, the URL will 404.
