        How to use this form
        1. Find your item. Type a part number, name, or description (e.g. WNR, wire nut, powder coat) and pick the matching result.
        2. Can't find it? Add it as a new item. Check "Add item not in database", name the item, choose Part / Supply / Other, and add a note for the purchaser if helpful.
        3. Flag urgency if needed. Check "Out of stock — urgent" only when we're truly out and it's blocking work. Keep this rare so real urgency stays visible.
        4. Add a note for the purchaser (optional). Use this for spec details, deadlines, links, or anything that helps them order the right thing.
        5. Add it to your request. Click Add. The item drops into "Your request" below. You don't pick quantity — the purchaser determines that based on annual usage, MOQ, and lead time. Repeat to add more items, then submit them all together.
        6. Pick your name and submit. Select yourself in Your name, optionally add a shared note that applies to every item, then click Submit purchase request.
        7. Track it. Use the Status tab at the top to watch each item move through Submitted → Waiting / Backordered → Ordered → Received.
        Tips
        • Bundle, don't split. One submission per shopping trip — add every item before clicking submit.
        • Quantity is the purchaser's call. Tell us what you need and why; they decide how many based on usage, MOQ, and lead time.
        • Notes get read. The purchaser sees your note alongside the item — write it the way you'd want it written for you.
