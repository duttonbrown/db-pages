How to use this page
  1. Pick your name. Select yourself in You are at the top so we know who signed for the shipment. Your name sticks across page loads.
  2. Find the order. Shipments are grouped by the supplier's Order # (the number on the packing slip). Use the search box for an item, vendor, PO, or order number; tap a supplier bubble to narrow to one vendor.
  3. Enter what actually arrived. The Received field is blank — type the qty that's in the box. The ordered qty shows as a faded hint next to it for reference. Don't guess. If the box is short, enter what's actually there — the next step handles the missing units.
  4. Flag any issues. Check the Issue box if anything's wrong (damaged, wrong part, missing parts inside the package) and add a quick note. The row still marks Received, but the purchaser sees the issue for follow-up.
  5. Confirm the shipment. Click Receive all to log every item in the group, or check the boxes you got and click Receive selected.
  6. Short shipment? Tell us if more is coming. If any qty you entered is less than what was ordered, a popup asks per item:
    • More expected (default) — this row closes at what arrived, and a new row appears for the remainder so the next box can be logged against it.
    • Nothing more coming — this row closes at what arrived, and the rest is written off. No new row.
Tips
  • One row = one box. Split shipments create sibling rows so each physical delivery has its own receiver, tracking, and date.
  • Order # matches the packing slip. If you can't find the row, search the order number printed on the slip — it's the most reliable match.
  • Tracking is on the row already. The purchaser fills it in at order time. You'll see it on the Status tab once shipments close.
  • Wrong qty? Fix it on the Status tab. Open the row in Notion from there if you typed the wrong number.
